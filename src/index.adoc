== O que é o Encontros Pythonicos?

Encontros Pythonicos é um projeto de estensão do curso de Ciência da Computação do departamento de informática da UERN/FANAT. +
Realizamos encontros periodicamente a cada mês para reunir a comunidade de Python local, tanto profissionais quanto entusiastas, para compartilhamos conhecimentos, dando aulas de diversos asuntos relacionados á Python, trocar experiências, esclarecer dúvidas e nos atualizar quanto ao mercado.
